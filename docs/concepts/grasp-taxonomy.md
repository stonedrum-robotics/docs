# Grasp Taxonomy

We support the Cutkosky grasp taxonomy for standardized manipulation tasks.

- **Power Grasps**: Cylindrical, spherical, palmar.
- **Precision Grasps**: Pinch, tripod, lateral.

You can trigger these directly using the `Hand.move_to_grasp("pinch")` method. See the [`GraspLibrary` API](../api-reference/grasp-library.md) for the full list of supported poses.
