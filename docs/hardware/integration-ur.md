# Universal Robots (UR) Integration

Mounting the hand to a UR5e or UR10e:

1. **Mechanical**: Use the standard ISO 9409-1-50-4-M6 adapter plate.
2. **Electrical**: Route the tool cable through the UR arm or use external dressing.
3. **ROS 2**: Launch the combined UR + Hand URDF for unified MoveIt 2 planning.
