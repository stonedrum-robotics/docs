# Linkerbot L20

The Linkerbot L20 is the flagship 21-DOF linkage-driven hand for research.

### Specifications
| Parameter | Value |
|-----------|-------|
| DOF | 21 (16 active + 5 coupled) |
| Weight | 1100g |
| Load | 10kg |
| Communication | CAN / RS485 |

### Wiring
```text
Pin 1: VCC (24V)
Pin 2: GND
Pin 3: CAN H
Pin 4: CAN L
```

**Note**: CE certification status is currently under verification — please contact Stonedrum Robotics for the latest compliance documentation.
