# MoveIt 2 Integration

The dexterous hand provides a full URDF/SRDF configuration package for MoveIt 2.

To start the MoveIt 2 demo:
```bash
ros2 launch dexterous_hand_moveit_config demo.launch.py
```

This launches RViz where you can use the interactive markers to plan collision-free motions for the hand joints or the arm.
