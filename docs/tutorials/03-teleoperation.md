# Teleoperation

This tutorial covers `examples/03_teleoperation_keyboard.py`.

It demonstrates how to map arbitrary inputs (like a keyboard or data glove) to joint targets in real-time.

1. The `TeleoperationInterface` allows defining mappings from sensors to joints.
2. In a loop, inputs are read, scaled, and sent as commands to the hand.

Please see the example script for the full implementation details.
